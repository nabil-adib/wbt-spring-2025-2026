# Task 2: Student Registration Form

Create a form where users can enter student information.

## Requirements

- **Insert element:** When the form is submitted, add the student details as a new row in a table.
- **Delete element:** Add a Remove button in each row to delete that student.
- **Update existing content:** Add an Update button to modify student name, age, or course.
- **Style manipulation:** Highlight rows when the mouse is over them, or color rows differently based on age/course.
- **Form validation:** Validate that:
  - Name is not empty
  - Age is a number
  - Course is selected

## Challenge

- Show validation errors below each input field.
